# Password Generator

A simple password generator web application built using HTML, CSS, and JavaScript.

## Features
- Generate random passwords
- Strong password generation
- Easy-to-use interface
- One-click password creation

## Technologies Used

- CSS
- JavaScript
- Git
- GitHub

password_generator/
